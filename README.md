# 생명과학 학습 노트

폰으로 읽는 생명과학 학습 자료. 세포라는 무대를 먼저 보고, 그 안에서 에너지가 어떻게 도는지 따라갑니다.

**바로 보기 →** https://hong-jame.github.io/cell-respiration/

### 세포라는 무대

| 편 | 내용 | 바로가기 |
|---|---|---|
| **1편 · 세포 구조** | 현미경·해상력, 표면적/부피, 원핵 vs 진핵, 내막계, 세포내공생설, 세포골격 | [열기](https://hong-jame.github.io/cell-respiration/cell-tour/) |
| **2편 · 막의 구조와 기능** | 유동 모자이크, 선택적 투과성, 삼투·팽압, 촉진확산, Na⁺/K⁺ 펌프, 집단수송 | [열기](https://hong-jame.github.io/cell-respiration/membrane/) |

### 에너지가 도는 법

| 편 | 내용 | 바로가기 |
|---|---|---|
| **3편 · 효소와 물질대사** | 자유에너지 ΔG, ATP, 활성화 에너지, 유도적합, 억제제, 되먹임 억제 | [열기](https://hong-jame.github.io/cell-respiration/enzymes/) |
| **4편 · 세포호흡과 발효** | 해당과정, 시트르산 회로, 전자전달사슬, 화학삼투, 발효, ATP 총결산 | [열기](https://hong-jame.github.io/cell-respiration/respiration/) |
| **5편 · 광합성** | 엽록체, 빛반응(광계·Z 도식), 캘빈 회로, 루비스코, 광호흡·C₄·CAM | [열기](https://hong-jame.github.io/cell-respiration/photosynthesis/) |

1·2편이 무대를 세우고, 3편에서 만든 효소가 4편에서 일하며, 4편이 분해하는 당은 5편이 만듭니다.

## 각 편의 구성

| 탭 | 내용 |
|---|---|
| **읽기** | 본문 6개 절 + 도해 7~10개 (탭하면 전체화면 확대) |
| **한눈에** | 한 장 요약 + 자주 틀리는 곳 |
| **퀴즈** | 12문항, 즉시 채점 + 해설 |
| **용어** | 25개 플래시카드 |

## 특징

- **각 편이 단일 HTML 파일 · 63~81KB** — 의존성 없음, 빌드 없음, 오프라인 동작
- **도해가 전부 인라인 SVG** — 벡터라 확대해도 선명하고, 다크모드에서 색이 함께 반전됨
- **모바일 우선** — 375px 기준으로 설계, 핀치 줌 지원
- 라이트/다크 모드 자동 대응

## 구조

```
index.html                목차
cell-tour/index.html      1편 · 세포 구조
membrane/index.html       2편 · 막의 구조와 기능
enzymes/index.html        3편 · 효소와 물질대사
respiration/index.html    4편 · 세포호흡과 발효
photosynthesis/index.html 5편 · 광합성
```

각 편의 헤더 ☰ 버튼으로 목차에 돌아옵니다.

## 저작권

본문과 도해는 모두 직접 작성·작도한 것입니다. 내용은 표준 생화학 교과 지식에 기반합니다.
