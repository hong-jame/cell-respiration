## 특징

- **단일 파일 · 78KB** — 의존성 없음, 빌드 없음, 오프라인 동작
- **도해가 전부 인라인 SVG** — 벡터라 확대해도 선명하고, 다크모드에서 색이 함께 반전됨
- **모바일 우선** — 375px 기준으로 설계, 핀치 줌 지원
- 라이트/다크 모드 자동 대응

## 다루는 내용

1. 에너지는 어디서 나오는가 — 산화환원, NAD⁺, 전자전달사슬
2. 해당과정 — 투자기/회수기, 순 ATP 2 + NADH 2
3. 피루브산 산화와 시트르산 회로 — 아세틸 CoA, NADH 6 + FADH₂ 2
4. 전자전달과 화학삼투 — 수소이온의 이동력, ATP 합성효소, ATP 총결산(약 30~32)
5. 발효와 무산소호흡 — NAD⁺ 재생, 알코올발효 vs 젖산발효
6. 물질대사의 교차로 — 이화작용 깔때기, 되먹임 조절
